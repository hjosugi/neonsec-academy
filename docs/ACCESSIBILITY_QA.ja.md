# アクセシビリティ QA

レビュー日: 2026-07-10

範囲: フェーズ6のビジュアルテーマ、エフェクトコントロール、レスポンシブレビュー、およびキーボードの使いやすさ。

## テーマプリセット

設定には4つのプリセットがあります：

| プリセット | エフェクト |
|---|---|
| ネオンナイト | フルサイバー雰囲気、スキャンラインオン、通常のコントラスト。 |
| ローグロー | ビジュアルスタイルを維持しながらネオンのぼかしを減少。 |
| ハイコントラスト | 明るいテキストと境界線、ローグロー、スキャンラインオフ。 |
| フォーカスモード | 動きの減少、ローグロー、高コントラスト、スキャンラインオフ、音オフ。 |

個別のトグルは、動きの減少、ローグロー、高コントラスト、スキャンライン、および音のために利用可能です。
音はデフォルトでオフです。

## チェックリスト

- [x] 質問、説明、ログ/コードのようなテキスト、報告フィールド、およびテーブルはテキスト優先のコントラストトークンを使用しています。
- [x] グリッチアニメーションはブランド/オンボーディングテキストに限定され、質問本文には適用されません。
- [x] 動きの減少モードは、ルートクラスを通じてCSSアニメーション/トランジションを無効にし、`prefers-reduced-motion`を尊重します。
- [x] ローグローモードは重いネオンのぼかしを取り除きます。
- [x] ハイコントラストモードは、 muted copy と hairlines を明るくします。
- [x] ステータスインジケーターは、色をテキスト、シンボル、ラベル、またはバッジと組み合わせます。
- [x] コマンドパレットは `/` または `Ctrl/Cmd+K` から開き、矢印/エンター/エスケープをサポートし、OSコマンドを実行しません。
- [x] レビューショートカットは `docs/KEYBOARD_SHORTCUTS.md` に文書化されており、回答選択、グレード、次、ブックマークをカバーしています。
- [x] モバイルレイアウトはサイドバー/グリッドを折りたたみ、レビューを使いやすく保ち、フルモックはデスクトップ推奨であることを警告します。
- [x] シティマップは二次的な進捗オーバーレイとして残り、非表示にすることができます。

## 既知のフォローアップ

- コミットされたブラウザベースのアクセシビリティスイートは、現在の手動チェックリストを繰り返し可能にします。
- バンドルコード分割は、v1受け入れランからのパフォーマンスフォローアップです。
