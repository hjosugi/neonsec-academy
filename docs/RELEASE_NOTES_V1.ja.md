# NeonSec Academy v1 リリースノート

現在のリリース: v1.0.5
リリース日: 2026-07-10

## ハイライト

- サイバーパンク CEH トレーナーには、オンボーディング、ダッシュボード、質問バンク、著作された質問、練習、間隔を空けたレビュー、分析、模擬試験、ファイナルゲート、安全なラボ、レポートビルダーが含まれています。
- ローカルファーストデータモデル: 進捗、設定、著作された質問、試験結果、間違い、レポートはブラウザに保存されます。
- 安全なラボワークフローは、ラボコンテンツが利用可能になる前に、合成された読み取り専用の証拠を使用し、可視のスコープ契約を伴います。
- 模擬試験フローには、ドメイン重み付けされたプリセット、アクティブな試験の再開/破棄、タイマー、フラグ付け、提出確認、結果レポート、回答レビューが含まれます。
- ファイナルゲートは、模擬ストリーク、期限のあるバックログ、弱いモジュール、在庫カバレッジ、回答カバレッジを要約し、Markdownとしてエクスポートします。
- レポートはラボからドラフトされ、Markdownとしてエクスポートできます。
- 設定には、動作の減少、低輝度、高コントラスト、スキャンライン、ラボスコアリングコントロール、公共安全なMarkdownエクスポート、完全バックアップJSONのインポート/エクスポート、質問パックのインポート/エクスポートが含まれます。
- オンボーディングでは、学習目標、日々のレビュー目標、オプションの目標日、シードバンクの好み、安全性の確認をキャプチャしてからアプリに入ります。
- コマンドパレットは、日々のレビュー、模擬試験、弱いモジュールのドリル、安全なラボ、レポートエクスポート、ファイナルゲート、練習、質問作成フローを起動します。
- 市の地図は、読み取り可能な地区ステータスバッジを表示し、レビューする必要のあるモジュールをルーティングし、コンパクトな隠し地図モードを含みます。
- テーマプリセットとアクセシビリティQA証拠は、低輝度、高コントラスト、動作の減少、ステータスラベル、コマンドパレットのキーボード使用、レスポンシブな試験履歴、都市地図のフォールバックをカバーします。
- 製品の範囲、分類法、データモデル、デザインシステムのワイヤーフレーム、情報アーキテクチャ契約、コンテンツ作成ガイダンスがPhase 0の基盤ドキュメントを締めくくります。
- 質問バンクは、タイトル、アーカイブ対応の詳細/編集フロー、マルチタグ/ステータス/結果フィルター、保存されたフィルター、自己採点の短答/レポートプロンプトの質問をサポートします。
- 質問の詳細は、タイムスタンプ、モード、結果、選択された回答、費やした時間、自信、総試行回数、正確性、最後に試行した時間を含む、質問ごとの試行履歴を表示します。
- 設定は、著作された質問をJSONパックまたはJSONL行としてエクスポートし、JSONパック、JSONL、または基本的なMCQ CSVを行番号付きの検証エラーと共にインポートできます。
- Markdownレンダリングは、エスケープされたHTML、コピーボタン付きのフェンコードブロック、パイプテーブル、安全なコールアウトをカバーします。
- リリースドキュメント、安全レビューの成果物、受け入れテストの証拠がリポジトリに含まれています。

## プライバシーと安全性

- 公共安全なMarkdownエクスポートは、集計された学習指標とマスクされたレポートタイトルのみを共有します。
- 完全バックアップJSONはプライベートであり、メモ、カスタム質問、回答、レポートの詳細を含む場合があります。
- 通常のアプリ使用には、バックエンド、テレメトリー、実際のターゲットスキャン、実際の資格情報、ライブマルウェア、またはサードパーティサービスとのインタラクションは必要ありません。

## 既知の制限

- 現在のプロダクションバンドルはViteの大きなチャンク警告を発生させています; コード分割はv1以降の最適化です。
- 受け入れスモークは、完全なタイムド125質問の実行ではなく、代表的な模擬フローとしてQuick Simを使用しました。
- 完全バックアップは個人の移行/回復にのみ適しており、公共の共有には公共安全なエクスポートを使用するべきです。
- デモコンテンツと専用のランディングモードはP6-010で追跡されています。
