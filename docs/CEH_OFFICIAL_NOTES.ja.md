# CEH公式ノート プランニング用

このIssue Packは、EC-Councilの公開情報をもとに構成しています。

## 主要な計画事実

- CEH v13 / CEH AIは20のモジュールを中心に構成される。
- 公式ページでは221のハンズオンラボ、550の攻撃手法、4,000以上のツールが説明されている。
- CEH知識試験は4時間/125の選択肢問題。
- CEH実践試験は6時間/20の実世界の課題。
- CEHマスターはCEH知識試験とCEH実践試験の両方を前提にする。
- CEHの典型的な合格カットスコアは試験形式により65%〜85%と説明されている。
