# Roadmap

## Phase 0: Foundation / Scope / Design

CEH 合格に必要な範囲、安全境界、情報設計、Cyberpunk Productivity UI の方向性を固定する。

### Critical Path

- P0-001: [P0-001] プロダクトビジョン・対象ユーザー・非ゴールを確定する
- P0-002: [P0-002] CEH 20 module と CEH+ 実践トラックの分類体系を定義する
- P0-003: [P0-003] 安全ポリシーと lab boundary を定義する
- P0-004: [P0-004] Question / Review / Attempt / Lab / Report のデータモデルを設計する
- P0-006: [P0-006] Cyberpunk Productivity Design System v0 を作る
- P0-007: [P0-007] 情報設計とナビゲーションを設計する

## Phase 1: Question Bank MVP

問題の作成・検索・回答・履歴保存ができる最小プロダクトを作る。

### Critical Path

- P1-001: [P1-001] アプリケーションの土台と開発ルールを作る
- P1-002: [P1-002] Question Bank CRUD を実装する
- P1-003: [P1-003] 複数の question type に対応する
- P1-004: [P1-004] CEH module / tag / difficulty で検索・フィルタする
- P1-006: [P1-006] 回答提出と即時 explanation 表示を実装する
- P1-007: [P1-007] Attempt history と基本統計を保存する
- P1-009: [P1-009] 安全な seed question set を作る
- P1-010: [P1-010] Core UI Layout を cyber terminal 風に実装する

## Phase 2: Review Queue / Retention

間違い・弱点・復習予定を管理し、毎日の復習を迷わず進められる状態にする。

### Critical Path

- P2-001: [P2-001] Review scheduler を実装する
- P2-002: [P2-002] Mistake Notebook を作る
- P2-003: [P2-003] Weakness Dashboard を作る
- P2-005: [P2-005] Daily Review Queue を作る
- P2-009: [P2-009] Review Session Summary を作る

## Phase 3: CEH Exam Mode

CEH の 20 module と 125 問試験形式に対応し、合格 readiness を測れる状態にする。

### Critical Path

- P3-001: [P3-001] CEH 20-module Coverage Matrix を作る
- P3-002: [P3-002] 125問・4時間の Mock Exam Mode を実装する
- P3-004: [P3-004] Exam Result Report を作る
- P3-007: [P3-007] Mock Exam Review Mode を実装する
- P3-008: [P3-008] Weak Module Drill を作る
- P3-010: [P3-010] CEH Final Gate Checklist を実装する

## Phase 4: Safe Practical Lab Mode

許可済みローカル教材だけで、CEH Practical 風の安全な hands-on 管理を実装する。

### Critical Path

- P4-001: [P4-001] Lab Registry と Scope Contract を実装する
- P4-002: [P4-002] Evidence Vault を作る
- P4-003: [P4-003] Flag Challenge Engine を実装する

- P4-006: [P4-006] Vulnerability Triage Workflow を実装する

- P4-009: [P4-009] CEH Practical-style 20 Challenge Simulator を作る
- P4-010: [P4-010] Lab Safety Audit を自動化する

## Phase 5: CEH+ Real-World Practical Track

### Critical Path

- P5-001: [P5-001] CEH+ Pentest Engagement Workflow Track を作る

## Phase 6: Cyberpunk Experience / Polish / Launch
