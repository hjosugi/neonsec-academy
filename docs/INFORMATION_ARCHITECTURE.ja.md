## スクリーン契約

| スクリーン | 目的 | メインエントリー | メインエグジット |
|---|---|---|---|
| ダッシュボード `/` | 次の学習アクションを選択し、準備状況、期限作業、ストリーク、弱点モジュールをスキャンする。 | アプリ起動、ナビゲーション、オンボーディング完了。 | レビューキュー、練習、模擬試験、分析、ファイナルゲート。 |
| 質問バンク `/bank` | 質問を検索、フィルタリング、ピン留め、アーカイブ、クローン、編集、検査する。 | ナビゲーション、コマンドパレット、ダッシュボードのクイックアクション。 | 質問詳細、新しい質問、モジュールまたはタグによる練習。 |
| 質問詳細 `/bank/:id` | 1つの質問を読み、回答し、説明を検査し、間違い/ブックマークの状態を管理する。 | 質問バンク、練習、レビュー、試験結果。 | 質問編集、次の練習、間違い、レビューキュー。 |
| 質問エディタ `/bank/new`, `/bank/:id/edit` | 検証とプレビューを伴う安全なローカル質問を作成または編集する。 | 質問バンク、質問詳細、設定インポートフロー。 | 保存時に質問詳細、キャンセル時に前の画面。 |
| 練習 `/practice` | すべての質問、モジュール、CEH+トラック、弱点エリア、または難易度による無制限のドリルを開始する。 | ダッシュボード、分析行クリック、シティマップ、ビヨンド。 | 質問ランナー、レビューキュー、分析。 |
| レビューキュー `/review` | 期限のある間隔反復アイテムをクリアし、記憶を評価する。 | ダッシュボードの期限カード、ナビゲーションの期限バッジ、コマンドパレット。 | 間違い、質問詳細、セッション概要、ダッシュボード。 |
| 模擬試験 `/exam` | タイムドCEH模擬試験を開始または再開し、試験履歴を検査する。 | ダッシュボード、ナビゲーション、コマンドパレット。 | 試験ランナー、試験結果、分析、ファイナルゲート。 |
| 試験ランナー `/exam/run` | 回答状態、フラグ、ナビゲーターを伴うタイムド試験を完了する。 | 模擬試験の再開/開始。 | 提出時に試験結果、キャンセル時に模擬試験。 |
| 試験結果 `/exam/result/:id` | スコア、合格/不合格状態、ドメインごとの結果、見逃した回答をレビューする。 | 模擬試験履歴、提出フロー。 | 弱点エリアの練習、質問詳細、分析。 |
| 分析 `/analytics` | モジュールマトリックス、ドメイン習得、フラグチャレンジ結果、準備要因、弱点モジュールを検査する。 | ダッシュボード、ナビゲーション、試験結果、セーフラボ。 | モジュールドリルの練習、セーフラボ、ファイナルゲート、質問バンク。 |
| 間違い `/mistakes` | 間違いノート、トラップパターン、次のアクション、解決済み状態を維持する。 | レビュー結果、質問詳細、ナビゲーション。 | 質問詳細、レビューキュー、練習。 |
| セーフラボ `/labs`, `/labs/:id` | 合成ラボを選択し、範囲を確認し、ローカル資産を検査し、フラグを提出し、記録されたヒントを表示し、説明を解除し、証拠を保存し、目標をスコアする。 | ナビゲーション、ビヨンドトラックリンク、ダッシュボード、分析。 | ラボ詳細、レポート、証拠ボールト、分析。 |
| 証拠ボールト `/evidence` | チャレンジごとにグループ化されたプライベート合成証拠をレビューし、編集のためにラボに戻る。 | ナビゲーション、ラボ詳細。 | ラボ詳細、レポート。 |
| CEH+トラック `/beyond` | 実践トラックをブラウズし、関連するドリルまたはラボを開始する。 | ナビゲーション、シティマップ、ダッシュボード。 | `track`での練習、セーフラボ、レポート。 |
| レポート `/reports` | 安全な調査結果レポートを作成し、ボールトの引用をリンクし、Markdownをエクスポートする。 | ラボ詳細、証拠ボールト、ナビゲーション、コマンドパレット。 | セーフラボ、証拠ボールト、設定エクスポート、ローカルMarkdownダウンロード。 |
| シティマップ `/map` | CEHモジュールとCEH+トラックを横断するセカンダリビジュアルナビゲーション。 | ナビゲーションワールドセクション、コマンドパレット。 | モジュールドリルの練習、ビヨンド、分析。 |
| 設定 `/settings` | テーマ/アクセシビリティの設定とローカルのインポート/エクスポートを管理する。 | ナビゲーション、コマンドパレット。 | ダッシュボード、インポート後の質問バンク、ローカルファイルエクスポート。 |
| ファイナルゲート `/final-gate` | スコア、バックログ、弱点モジュール、カバレッジに対する事前予約の準備状況を確認する。 | ダッシュボード、分析、模擬試験。 | 模擬試験、レビューキュー、分析、Markdownエクスポート。 |

## ディープリンク

| リンク | 動作 |
|---|---|
| `/bank/:id` | 特定の質問を開く。 |
| `/bank/:id/edit` | 特定のローカルまたはクローンされた質問を編集する。 |
| `/practice?module=5` | CEHモジュール5にスコープを絞った練習を開始する。 |
| `/practice?track=soc` | CEH+ SOCトラックの練習を開始する。 |
| `/exam/run` | アクティブな試験セッションが存在する場合は再開する。 |
| `/exam/result/:id` | 提出された試験結果を開く。 |
| `/labs/:id` | 特定の合成ラボを開く。 |
| `/evidence` | ラボチャレンジごとにグループ化された証拠ボールトを開く。 |

## 空の状態

| エリア | 空の状態アクション |
|---|---|
| レビューキュー | 練習を開始するか、質問に回答してレビューアイテムを作成する。 |
| 試験履歴 | クイックシミュレーションまたはフル試験を開始する。 |
| 間違い | 弱点モジュールを練習する; ノートが保存された後に間違いが表示される。 |
| レポート | セーフラボから開始するか、空の合成レポートを作成する。 |
| 質問バンクのカスタムフィルタ | フィルタをクリアするか、カスタム質問を作成する。 |

## ショートカット

| ショートカット | アクション |
|---|---|
| `/` または `Ctrl/Cmd+K` | コマンドパレットを開く。 |
| 矢印キー | コマンドパレットオプション内を移動する。 |
| Enter | 選択したコマンドをアクティブにする。 |
| Escape | コマンドパレットまたはモーダルサーフェスを閉じる。 |

## 主要ユーザーフロー

### デイリーレビュー

```text
ダッシュボード → レビュー開始 → 回答 → 説明 → 自信 → 概要 → 次のアクション
```

### 模擬試験

```text
試験 → プリセット選択 → 125質問 → 提出 → 結果レポート → 間違った回答をレビュー
```

### 実践ラボ

```text
ラボ → スコープ契約 → ローカル資産 → フラグ試行 / ヒント → 説明 → 証拠 → 調査結果 → レポート → 分析
```
