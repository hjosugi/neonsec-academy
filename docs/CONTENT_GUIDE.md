# Content Authoring Guide

## Question Checklist

- [ ] 1 問 1 concept になっている
- [ ] CEH module が付いている
- [ ] difficulty が妥当
- [ ] explanation に why / trap / memory phrase がある
- [ ] 実ターゲットや実攻撃手順を含まない
- [ ] 間違い選択肢が雑ではない

## Lab Checklist

- [ ] scope contract がある
- [ ] synthetic/local dataset だけを使う
- [ ] forbidden action が明記されている
- [ ] evidence を要求する
- [ ] remediation を要求する
- [ ] report に転用できる

## Good Explanation Pattern

```text
Answer:
This is correct because ...

Why others are wrong:
A is wrong because ...

Trap:
The common trap is ...

Memory phrase:
Short phrase to remember.
```
