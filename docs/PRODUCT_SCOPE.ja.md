# プロダクトスコープ

NeonSec Academyは、CEH学習者が質問を管理し、間違いを見直し、安全に実践的な推論を証明するのを助けるために存在します。これはローカルファーストの学習製品であり、攻撃ツール、スキャナー、またはホスティングされたトレーニングプラットフォームではありません。

## ペルソナ

| ペルソナ | ニーズ | プロダクトフィット |
|---|---|---|
| プライマリー: CEH候補者 | すべての20のCEHモジュールにわたって十分な幅を持ち、弱い概念を保持し、試験のタイミングをリハーサルする。 | 質問バンク、間隔を空けたレビュー、模擬試験、準備スコアリング、最終ゲートチェックリスト。 |
| セカンダリー: 初期キャリアのセキュリティ実務者 | 実際のシステムに触れることなく、防御的分析、報告、優先順位付けを実践する。 | 合成安全ラボ、レポートビルダー、CEH+トラック、公共安全な進捗エクスポート。 |
| セカンダリー: コンテンツ著者またはメンター | プライベートな資料を漏らすことなく、安全なカスタム質問と再利用可能な学習パックを作成する。 | 検証済みのローカル著作、質問パックのインポート/エクスポート、コンテンツ安全ガイダンス。 |

## ユースケース

1. 日常的な学習: ダッシュボードを開き、期限のある質問をレビューし、弱いモジュールを練習し、間違いを記録する。
2. 試験リハーサル: タイマー付きの模擬試験を開始し、回答を提出し、ドメインごとの結果を確認し、ギャップを掘り下げる。
3. 安全な実践練習: ラボスコープ契約を認識し、合成証拠を分析し、発見を書き、レポートをエクスポートする。
4. コンテンツ著作: オリジナルの質問を作成またはインポートし、メタデータを検証し、公共安全な質問パックのみを共有する。
5. 進捗の引き継ぎ: 生の回答、プライベートノート、証拠、または個人の学習履歴なしで、集計分析または最終ゲートのステータスをエクスポートする。

## MVPスコープ

MVPには以下が含まれます：

- すべての20のモジュールをカバーするCEHマッピングされたシード質問。
- 検索、フィルタ、ピン留め、クローン、アーカイブ、およびカスタム質問の著作。
- 選択肢、複数選択、真偽、短答、シナリオ、およびレポートプロンプトの練習フロー。
- 即時採点、説明、試行履歴、間違いノート、および間隔を空けたレビュー。
- タイマー付き模擬試験、モジュール/ドメイン分析、準備スコアリング、および最終ゲートチェック。
- 合成の読み取り専用安全ラボ、レポート練習、およびCEH+トラックナビゲーション。
- ローカルJSONバックアップとユーザー著作の質問パックのインポート/エクスポート。

MVPには以下は含まれません：

- ホスティングアカウント、同期、リーダーボード、コラボレーション、支払い、またはテレメトリー。
- 実際のターゲットスキャン、ライブエクスプロイト実行、フィッシング配信、パスワードクラッキング、マルウェア、またはトラフィック生成。
- 公式EC-Councilコンテンツ、試験ダンプ、著作権で保護された質問クローン、または合格保証の主張。
- 完全なSIEM、脆弱性スキャナー、攻撃ラボ、資格情報ボールト、またはプロダクションインシデントプラットフォーム。

## 成功指標

| 指標 | 目標シグナル |
|---|---|
| CEHモジュールカバレッジ | すべての20のモジュールに利用可能な質問と可視カバレッジステータスがある。 |
| 学習保持 | 期限のあるレビューのバックログが減少し、正確性と習得度が上昇する。 |
| 試験準備 | 模擬試験のスコアが設定された目標を満たし、弱いモジュールの数が最終ゲートの閾値を下回る。 |
| 安全な実践準備 | ラボレポートにはスコープ、証拠、影響、修正、および明示的な合成境界が含まれる。 |
| プライバシーと安全 | エクスポートは生の回答、レポート証拠、個人ノート、実際の資格情報、およびライブターゲットの詳細を避ける。 |

## 安全境界

すべての組み込みコンテンツは、ローカル、合成、架空、または文書範囲の資料を使用する必要があります。実際のシステムとの相互作用を必要とする質問、ラボ、レポート、またはインポートはスコープ外であり、拒否されるか、防御的アーティファクト分析として書き直される必要があります。運用ポリシーは[安全境界](SAFETY_BOUNDARIES.md)に維持されています。
