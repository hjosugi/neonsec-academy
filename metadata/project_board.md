# Project Board

## Columns

1. Inbox
2. Ready
3. In Progress
4. In Review
5. Blocked
6. Done

## Views

- By Phase
- By Priority
- By Area
- Current Sprint
- Safety Review
- Release Readiness

## Rules

- `priority:P0` は Phase ごとの critical path。
- `type:security` は release 前に必ず review。
- `area:content` は QA checklist を通す。
- `area:lab` は Safety Audit を通す。
