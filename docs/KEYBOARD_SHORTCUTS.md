# Keyboard Shortcuts

## Global

| Shortcut | Action |
|---|---|
| `/` | Open command palette |
| `Ctrl+K` / `Cmd+K` | Open command palette |
| `Esc` | Close command palette or active overlay |

## Practice And Review

| Shortcut | Action |
|---|---|
| `1`-`9` before reveal | Pick the matching answer choice |
| `Enter` before reveal | Submit or reveal the model answer |
| `B` | Toggle bookmark |
| `N` / `Right Arrow` after practice reveal | Go to the next practice question |
| `Enter` after practice reveal | Go to the next practice question |

## Review Grading

| Shortcut | Action |
|---|---|
| `1` after reveal | Grade `Again` |
| `2` after reveal | Grade `Hard` |
| `3` after reveal | Grade `Good` |
| `4` after reveal | Grade `Easy` |
| `Enter` / `N` / `Right Arrow` after reveal | Grade `Good` if correct, `Again` if incorrect |

Shortcuts are ignored while typing in form fields.
