# ラベル

- `phase:0`: 基盤、範囲、安全性、デザインの方向性
- `phase:1`: 質問バンク MVP
- `phase:2`: レビューキューと保持
- `phase:3`: CEH試験モード
- `phase:4`: 安全な実践ラボ
- `phase:5`: CEHプラス実践トラック
- `phase:6`: サイバーパンクの仕上げとローンチ
- `type:epic`: 大規模な親レベルの作業
- `type:feature`: ユーザー向け製品機能
- `type:content`: 質問、ラボ、または学習コンテンツ
- `type:design`: ビジュアル、UX、またはインタラクションデザイン
- `type:security`: 安全性、セキュリティ、法的、及び悪用防止
- `type:infra`: ツール、データ、インポート、エクスポート、CI
- `type:quality`: テスト、QA、メトリクス、リリース準備
- `area:question-bank`: 質問の保存、作成、検索
- `area:review`: レビューキュー、間隔反復、間違い
- `area:exam`: 模擬試験と準備
- `area:lab`: 安全なローカルラボとチャレンジ管理
- `area:design`: サイバーパンクUIとアクセシビリティ
- `area:analytics`: 統計、ダッシュボード、レポート
- `area:content`: CEHコンテンツとCEH+コンテンツ
- `area:safety`: 認可の境界と有害使用防止
- `priority:P0`: 重要な経路
- `priority:P1`: 重要
- `priority:P2`: コアフローの後にあれば良い
