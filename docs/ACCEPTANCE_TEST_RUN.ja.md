## 安全性とプライバシー

- ラボは合成の読み取り専用素材の中に留まりました。
- スコープ契約は、ラボコンテンツが利用可能になる前に禁止されたアクションを明示的に表示しました。
- パブリックセーフエクスポートは、生の回答、カスタム質問テキスト、間違いノート、レポート証拠、完全バックアップデータを除外します。
- パブリックセーフエクスポートは、現在のプライバシーエクスポートヘルパーを通じて一般的な敏感なレポートタイトルパターンをマスクします。
- 完全バックアップ JSON は設計上プライベートであり、ローカルノート、回答、カスタム質問、レポート詳細を含むとラベル付けされています。
