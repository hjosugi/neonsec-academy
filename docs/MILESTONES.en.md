# Milestones

## Phase 0 - Foundation

Fix the scope necessary for CEH certification, safety boundaries, information design, and the direction of the Cyberpunk Productivity UI.

## Phase 1 - Question Bank MVP

Create a minimum product that allows for the creation, search, answering, and history saving of questions.

## Phase 2 - Review System

Manage mistakes, weaknesses, and review schedules to enable daily reviews without hesitation.

## Phase 3 - CEH Exam Mode

Support the 20 modules and 125-question exam format of CEH, allowing for the measurement of exam readiness.

## Phase 4 - Safe Practical Labs

Implement safe hands-on management in a CEH Practical style using only approved local materials.

## Phase 5 - CEH Plus Practical Track

Enhance practical skills in pentesting / AppSec / cloud / SOC / IR beyond CEH using safe materials.

## Phase 6 - Cyberpunk Polish and Launch

Refine the experience as a cyberpunk experience that does not interfere with question management and review management, achieving v1.0 release quality.
