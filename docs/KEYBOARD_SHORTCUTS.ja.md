# キーボードショートカット

## グローバル

| ショートカット | アクション |
|---|---|
| `/` | コマンドパレットを開く |
| `Ctrl+K` / `Cmd+K` | コマンドパレットを開く |
| `Esc` | コマンドパレットまたはアクティブオーバーレイを閉じる |

## 練習とレビュー

| ショートカット | アクション |
|---|---|
| `1`-`9` リビール前 | 一致する回答選択肢を選ぶ |
| `Enter` リビール前 | モデル回答を提出または表示する |
| `B` | ブックマークを切り替える |
| `N` / `右矢印` 練習リビール後 | 次の練習問題に進む |
| `Enter` 練習リビール後 | 次の練習問題に進む |

## レビューの採点

| ショートカット | アクション |
|---|---|
| `1` リビール後 | `再度` として採点 |
| `2` リビール後 | `難しい` として採点 |
| `3` リビール後 | `良い` として採点 |
| `4` リビール後 | `簡単` として採点 |
| `Enter` / `N` / `右矢印` リビール後 | 正解なら `良い` として採点、不正解なら `再度` として採点 |

ショートカットはフォームフィールドに入力中は無視されます。
